# Bound

Утилита для синхронизации полей класса с его mobx объектом.

## Использование

```bash
 python bound.py [-s | --save_state] [-h] path
```

+ ### Синхронизация полей с сохранением изначального порядка.
    ```bash
      python ./bound.py -s ~/path_to_ts_file.ts
    ```
+ ### Помощь
    ```bash
      python ./bound.py -h
    ```
